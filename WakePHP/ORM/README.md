How to add ORM class (data object)
==================================

1. Let's take name of collection as ExternalSignupRequests  
1. In namespace \WakePHP\ORM we should create file ExternalSignupRequests.php with class ExternalSignupRequests   
1. Class ExternalSignupRequests should be inherited from \WakePHP\Core\ORM    
1. In next launch WakePHP will scan folder ORM and load objects.
